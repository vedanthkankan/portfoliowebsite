My Works:

  1. Movie Ratings Prediction DataSet:

    In this project, I developed a movie rating prediction model using the IMDB Movies Dataset. By employing Python and libraries like Pandas, Matplotlib, and Seaborn, I explored the dataset to understand key features such as genre, director, and release year. After preprocessing the data and encoding categorical variables, I trained and evaluated various regression models to predict movie ratings. The project involved visualizing data distributions and model performance, ultimately providing insights into how different factors influence movie ratings and identifying areas for model improvement.

  2. Sales Prediction DataSet:

    In my Sales Prediction Data Science project, I utilized an advertisement dataset to forecast product sales by analyzing the impact of advertising expenditure, target audience segmentation, and advertising platform selection. Leveraging Python and libraries such as pandas, NumPy, matplotlib, seaborn, and scikit-learn, I developed a predictive model to estimate future sales. The project involved data preprocessing, exploratory analysis, feature engineering, and model training, resulting in actionable insights that can help optimize advertising strategies and enhance sales forecasting accuracy.

  3. Titanic Survival Prediction DataSet:

    In my Titanic Survival Prediction Data Science project, I analyzed the passenger dataset to predict survival outcomes. Using Python and libraries like pandas, NumPy, matplotlib, seaborn, and scikit-learn, I developed a model to determine whether passengers survived based on features such as age, sex, class, and fare. The project involved data preprocessing, exploratory data analysis, feature engineering, and model evaluation, ultimately enabling accurate predictions of survival probabilities and providing insights into key factors influencing survival on the Titanic.
